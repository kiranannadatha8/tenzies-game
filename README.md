> A fast-paced React implementation of the classic Tenzies dice game. Roll until all dice show the same value — hold the ones you want to keep between rolls.

**[▶ Play live →](http://tenzies-game-fawn.vercel.app)**

---

## 🎮 How to play

1. Roll the dice
2. Click any die to **hold** it between rolls (it turns green)
3. Keep rolling until **all dice show the same number**
4. Win 🎉

---

## ✨ Features

- One-click dice rolling with animated state updates
- Hold/unhold individual dice with visual feedback
- Win detection with confetti celebration
- Roll counter to track efficiency
- Clean, responsive UI — works on mobile and desktop

---

## 🛠️ Built with

- **React** — component-based UI, useState / useEffect hooks
- **nanoid** — unique key generation for dice
- **react-confetti** — win celebration animation
- **CSS3** — custom styling, flexbox layout, transitions
- **Vercel** — zero-config deployment

---

## 🚀 Run locally

```bash
git clone https://github.com/kiranannadatha8/tenzies-game
cd tenzies-game
npm install
npm start
```

## 💡 What I learned

Deep-dive into React fundamentals — lifting state, deriving computed values from state arrays, and managing side effects with useEffect for win detection. The hold/unhold mechanic required understanding immutable state updates with .map().
